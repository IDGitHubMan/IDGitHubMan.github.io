Python

Python is an interpretive, object-oriented programming language that is mostly used for data processing and machine learning. Python lacks the visualization simplicity of Processing, which is actually why I went on to learn processing.

I set out to learn it because it was a simple way to get back into programming and development after a long hiatus. I have mostly made little tools with it. I attempted starting a game, but that proved difficult in tkinter and Python in general. I decided to mess around with matplotlib and turtle for a period, making a whole bunch of sequence graphing programs, and some artsy stuff. Alongside these, I have some sort of helpful math tools.
